# Gallery UI

Web UI for MCP article, built with Next.js.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Configure Environment
cp env.example .env
# Update NEXT_PUBLIC_API_URL in .env.local

# 3. Development Mode
npm run dev
```
